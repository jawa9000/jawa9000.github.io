Super Cave Boy

Simple yet challenging at times game based off a tutorial I completed in January 2017.

Platform game with several levels and a few challenging enemies. Can you make it to the final level?

Controls

  • Move: left and right arrow keys
  • Jump: up arrow key
  • Enter door: down arrow key
  • Restart room: R key
