Expandables with Rotating Indicators

This instructive guide assumes the user has basic understanding of JavaScript, jQuery, and basic cookie manipulation.

Written by Brian Immel

HTML Update: Define Character Set

Updated the header with this line:

...
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
...

What does this do?

<meta charset="UTF-8"> is used to set the characeter set on an HTML document. Without this character set, the arrow character used in this tutorial ("▶") wouldn't show up properly. Setting the character set to "UTF-8" allows the browser to read in CSS files as this format. The UTF-8 format allows for special character sets such as unicode and ASCII.

HTML Update: Span.arrow

With the character set to use special characters, the next thing that needs to be updated is the span element nested within the expandable's title element. This span element will receive the class of arrow.

...
<span class="arrow" title="Click to expand">Expandable Title</span>
...
CSS Additions

Add three CSS rules: .arrow, .arrow:before, .arrowOpen:before

We need to add a class for the span element that will do the following:

  • set up a relative position for the arrow reside in
  • make the arrow slightly larger than it's default size
  • Make some space to the left of the content so the arrow doesn't overlap the expandable's title
.arrow {
position: relative;
display: block;
color: #4891dc;
font-size: 1.1em;
padding: 0 0 0 30px;
}
Arrow Defaults

To create the rotating arrow, we are going to use the span.arrow's relative position to absolutely position the arrow using the content property in a before psuedoclass. To get the animated effect, we set a transform (initially at zero) with a transition value of 0.4 seconds. This transition doesn't do anything until a jQuery event is used to add/remove the .arrowOpen class. This class rotates the arrow 90 degrees (pointing down). This would indicate that the expandable has been opened. Otherwise, when its pointed to the right, its pointing to the content's location

.arrow:before {
display: block;
content: '▶';
transform: rotate(0);
transition: 0.4s;
position: absolute;
left: 0px;
top: -2px;
}
Rotating 90 Degrees

With the arrow's default behavior set, we need to create the transform effect when the user clicks on the span element. In this case, the arrow will rotate 90 degrees (to point down).

.arrowOpen:before {
transform: rotate(90deg);
}
Hover Effect

On the final bit of CSS, we add a little hover effect that is similar to links: changing colors. In this case, since we have a transition in effect, the color change happens gradually instead of the usual instant color change. If you don't like this effect, you can leave this rule out.

span.arrow:hover {
color: white;
}

jQuery Additions

Add two lines of jQuery in the click event for the expandable

And now, to update the jQuery side of this feature, we just need to add two lines of code to trigger the rotation of this arrow indicator when the user clicks the expandable's title element.

...
$("#" + titleId + " span").addClass("arrowOpen");
...
...
$("#" + titleId + " span").removeClass("arrowOpen");
...
Complete jQuery
Complete jQuery
$(document).ready(function() {
$("div[id^='expandTitle']").click(function() {
var titleId = $(this).attr("id");
var siblingId = $("#" + titleId + " ~ div").attr("id");
var status = $(this).attr("status"); //open/hidden
if (status == "hidden") {
if ($("#" + titleId + " span").hasClass("arrow")) {
$("#" + titleId + " span").addClass("arrowOpen");
}
$("#" + siblingId).slideDown("fast");
$(this).attr("status","open");
} else {
if ($("#" + titleId + " span").hasClass("arrow")) {
$("#" + titleId + " span").removeClass("arrowOpen");
}
$("#" + siblingId).slideUp("fast");
$(this).attr("status","hidden");
}
});
});
