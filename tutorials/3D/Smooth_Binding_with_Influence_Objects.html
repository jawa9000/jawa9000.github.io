Smooth Binding with Influence Objects

Written by Brian Immel

Create Influence Object

Create an influence object that will fit the shape and design of the area to be deformed. In this case, I'm using a simple poly sphere with a 6 x 6 subdivision that will roughly fit the area of the shoulder.

Select the influence object and duplicate it. Offset the duplicated object so it doesn't sit on top of the original object.

Name the duplicated influence object according the name of the direction it will help deform (shoulder_up, shoulder_down, etc.) as we did before in the Indirect Rigid Binding tutorial.

Blend Shape Relationship

Create a Blend Shape relationship between the base and morphing object.

Create a SDK relationship between the Blend Shape of the influence object and the rotation of the shoulder joint as we did before in the Indirect Rigid Binding tutorial.

Edit Smooth Skin

Select the influence object and parent it to the joint nearest the area you want to influence but do not parent an influence object to a joint that rotates. This can cause a huge headache if that joint rotates heavily (like shoulders do). In this case, select the joint just before the shoulder/armpit joint and parent it to that one.

Deselect everything and select the smooth bound geometry first.

Shift-select the influence object and go to Skin > Edit Smooth Skin > Add Influence.

There are several ways to set up the influence object. Some methods involve using the Expression Editor to drive the shape of the influence object(s) other use MEL as well to operate the influence object's scale, rotation and shape. Of the three mentioned techniques, using Blend Shapes is the simplest (especially if you're not too good with scripting) but Expressions would normally evaluate faster.
