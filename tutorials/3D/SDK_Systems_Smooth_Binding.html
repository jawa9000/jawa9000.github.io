Set Driven Skeleton Systems With Smooth Binding

The first thing we need to do is set up the positions for branching off the shoulder to simulate muscle movements below the skin.

Branching

Set up a series of joints surrounding the shoulder joint. Four joints to build the basic branch:

Deltoid (top of shoulder). Keep in mind that the deltoid muscle is the primary mover of the shoulder when setting up the SDK of the branched joints. This joint will have a very crucial roll in the displacement of the shoulder geometry.

Many muscle groups influence the armpit area but the majority of these groups are influenced mostly by the primary muscle groups that surround the shoulder joint. Some of these groups include deltoid, pectoralis major and latissmus dorsi. Some of the secondary muscle groups include the teres major and coracobrachialis. Because there are so many muscles influencing the armpit, we'll generically name the joint assigned to handle that area simply put as armpit.

Pectoralis (front of shoulder)

Scapula (back of shoulder).

Set Driven Key the joints surrounding the shoulder joint

Use a controller object to drive the branch joints. In this case, select the S curve named Shoulder_Control_L.

Open up the SDK window and load the Shoulder_Control_L as the driver. Select the attribute Shoulder_UD.

Set the Deltoid Joint

Select the deltoid joint and load it as the driven.

All the shoulder branches will be driven via their translate attributes. Select the translate attributes of the deltoid joint.

Hit the Key button to set the default zero position.

SDKing the Deltoid

Change the value of the attribute Shoulder_UD to somewhere between 1 and 9.

Translate the deltoid joint somewhere between the middle of the joint's movement and hit Key in the SDK window.

Repeat this step and the last one a few times to set up a smooth transition between zero movement and total movement of the deltoid joint. I usually set up three in-between SD keys between the values of 0 and 10.

Deltoid Bent Up

Change the value of the attribute Shoulder_UD to 10.

Translate the deltoid joint to accommodate the folds of a shoulder being bent up.

Select the translate attributes of the deltoid joint in the SDK window and hit Key.

Shoulder Down

Change the value of the attribute Shoulder_UD to somewhere between -1 to -9.

Translate the deltoid joint somewhere between the middle of the joint's movement and hit Key in the SDK window.

Repeat this step a few times to set up a smooth transition between zero movement and total movement of the deltoid joint. I usually set up three in-between SD keys between the values of 0 and -10.

Deltoid Folds

Change the value of Shoulder_UD to -10.

Translate the deltoid joint to accommodate the folds and pulls of a shoulder being folded down.

Select the translate attributes of the deltoid joint in the SDK window and hit Key.

Select the armpit joint and repeat steps 2-7 replacing deltoid joint with armpit joint. Continue this steps with all the branch joints until they are all driven. Note: these SDKs are subject to be tweaked once the geometry has been skinned. It may be desirable to skin the geometry first and then set up the SDK of the branch joints.

Parting Notes

When it comes to painting weights, the majority of shoulder geometry weighed to the branches (if not all).

Branch end joints move (SDK) according to muscular movements of the shoulder. This may take some R&D time to get the movements you like. Setting up a SDK branch joint setup will require a certain amount of balance between the movements of the joint ends and the weights applied to them.
