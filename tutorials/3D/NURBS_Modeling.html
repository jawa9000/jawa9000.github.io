NURBS Modeling

I will be talking about various methods for modeling with NURBS including creating NURBS curves, creating surfaces from curves and modifying NURBS surfaces.

Written by Brian Immel

Overview

This workshop will cover all the basic functions and uses for NURBS modeling from using a simple curve to a highly detailed surface.

NURBS objects can be thought of as pieces of paper folded to make the shape you want. The reason the comparison between NURBS surfaces and paper is that a NURBS surface can be "wrapped" like paper.

Object History, Freezing Transformations and Centering Pivots
  • At some point in time when modeling or setting up for animation, it will become necessary to delete the history of various objects for a wide variety of reasons.
    • Edit > Delete by Type > History
    • I use this function so often that I make it a shortcut on my Shelf.
  • Freezing Transformations is helpful to both the modeler and animator. Sometimes you will need to 'reset' or modifiy a transformation node.
    • Modify > Freeze Transformation
    • Again, I use this function frequently so I make it a shortcut on my Shelf.
  • Centering Pivots, as the name suggests, centers the pivot of an object to its averaged center position.
    • Modify > Center Pivot
    • I also have this function in my Shelf because its so handy.
