Hyperloop 3.0.0 Release Note

Archive note: all links and images have been disabled as their destinations and/or source locations no longer exist. The main navigation is also missing.

Created by Brian Immel, last modified on Nov 17, 2017

Hyperloop 3.0.0 is a major release that includes new features, improvements, and bug fixes.

New features

  • TIMOB-24657 - Hyperloop: Android - Support to catch native exceptions
    • Added feature that catches native exceptions

Improvements

  • TIMOB-25340 - Hyperloop: Move to using auto-required 'hooks' folder, not plugin
    • As part of the integration of Hyperloop into SDK core, the <plugins> tag is no longer needed for Hyperloop
  • TIMOB-25399 - Windows: Hyperloop to drop Windows 8.1 support
    • Removed support for Windows 8.1

Fixed issues

  • TIMOB-25079 - Use JavaScriptCore by default
