Studio 5.3.0.RC

Studio 5.3.0.RC is a patch release that includes new features, improvements, and bug fixes.

  • Updated grammar files for JavaScript and generated a new ES6/7 compliant parser for Studio

Improvements

  • Importing app into studio restructures tiapp.xml unnecessary

Fixed issues

  • TISTUD-9031 - Studio installs outdated hyperloop plugin
  • TISTUD-9042 - iOS: Create modules in "dist" directory
  • TISTUD-9074 - Studio unable to load provisioning profiles after SDK/CLI upgrade
  • TISTUD-9090 - Appc studio encounters java.lang.NullPointerException in 5.1.0.201806130758
  • TISTUD-9092 - Error occurred during "Getting Diagnostic Logs"
  • TISTUD-9099 - Incorrect filename shown in message on the error marker
  • TISTUD-9101 - Install android 8.1 by default in the appcelerator studio

Known issues

  • No know or reported issues for version 5.3.0
